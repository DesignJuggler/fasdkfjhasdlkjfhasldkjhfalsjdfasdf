Grid MP Features and Components

Grid MP has been named "Best Price-Performance Grid Middleware Solution" and "Most Innovative Grid Management or Performance Improvement Software."

Innovative and cost-effective. It doesn't get much better than that.

Grid MP features include:

  • Uniquely engineered for non-dedicated HPC: Grid MP is the only technology designed from the ground up for large-scale, disparate non-dedicated environments where compute devices are not tightly controlled – so unpredictability is accounted for
  • Proven, unparalleled scalability: Grid MP was built specifically for environments of massive scale – the product regularly scales to thousands of production customer nodes
  • Simple to operate and manage: With a robust, web-based GUI, decentralized management of resource allocation policies, and automatic software update features for all code and program modules, Grid MP is easy for customers to use and manage
  • Fast, easy application on-boarding: Grid MP features a flexible, open, standards-based Web services interface for integrating applications, and includes a Software Development Kit (SDK) with detailed documentation, tools and examples. Scores of commercial applications have been integrated to run on Grid MP.
  • Embedded meta-data layer for automatic tracking, reporting and validation: Grid MP offers detailed reporting and monitoring of resources, users and jobs

Grid MP components include:

Component Description Benefits
Virtualization, Provisioning and Control
  • Single virtual interface to access SMP, cluster, server, workstation and desktop devices
  • Resource grouping based on capability, geography & organization
  • Decentralized management of resource allocation policies
  • Ability to maintain control of environment
  • Optimized resource allocation for better performance
  • Improved manageability
Secure Execution and Automation Layer (SEAL) Technology
  • Secure runtime environment protects application execution from tampering and interception
  • Automatic encryption and compression of data over the network and during storage (for non-dedicated resources)
  • Digital signature validation of executable code modules
  • Protection for users, applications and data
  • System reliability
Advanced Workload Optimization
  • Scheduling based on job priority, resource constraints, fair share, and more
  • Automatic job rescheduling in the presence of failures
  • Scheduling based on affinity for cached data
  • Application checkpointing for improved workload processing performance
  • Optimized performance and throughput
  • Automation for reduced management overhead
  • Maximized ROI on existing resources
Industry-Leading Application Framework
  • Simple command-line launch of sequential batch or tightly-coupled parallel (MPI, PVM, etc.)
  • Open, standards-based Web services interface for integrating applications
  • End-user transparency – application users continue to use the same interface
  • Software Development Kit (SDK) with detailed documentation, tools and examples
  • Application scripting can be done using several different languages and operating systems
  • Fast, easy application on-boarding
  • Ease of application use and management
  • Improved application utilization and value
Seamless Integration with Enterprise Environments
  • Scalable component architecture capable of concurrently handling thousands of devices and jobs
  • Automatic software update features for all code and program modules
  • Web-based management console
  • Detailed reporting and monitoring of resources, users and jobs
  • Support for Internet Protocol Version 6 (IPv6)
  • Broad support for a wide range of web browsers and agent operating systems
